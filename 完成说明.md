## ✨ 动画效果汇总

### 1. 页面加载（自动播放）
```
时间线：
0.0s   → 导航栏就位
0.2s   → 头像淡入 + 浮动
0.3s   → 标题滑入
0.4s   → 副标题淡入
0.5s   → 描述淡入
0.6s   → 按钮淡入
```

### 2. 悬停交互（即时响应）
```
卡片：      向上浮起 (-10px) + 阴影增强
按钮：      缩放 (1.02) + 上移 (-3px) + 发光
标签：      弹跳 (1.1x) + 背景变化
输入框：    焦点放大 (1.02x) + 边框发光
```

### 3. 滚动动画（性能优化）
```
进入视口：  淡入 + 上升 (30px → 0px)
离开视口：  保持最终状态
返回顶部：  平滑滚动 + 动画
导航栏：    阴影动态变化
```

### 4. 动画参数
```
缓动曲线：  cubic-bezier(0.34, 1.56, 0.64, 1)
特点：      超越终点 → 产生弹性感
时间：      0.4 秒（平衡 - 不快不慢）
毛玻璃：    blur(20px) saturate(180%)
```

## 🔧 常见定制

### 改变主色调
```css
/* 在 assets/styles.css 第 10-20 行 */
:root {
    --primary-color: #你的颜色;
}
```

### 调整动画速度
```css
/* 在 assets/styles.css */
--transition: all 0.3s ...;  /* 更快 */
--transition: all 0.5s ...;  /* 更慢 */
```

### 启用/禁用毛玻璃
```css
/* 注释掉下面的行 */
/* backdrop-filter: blur(20px) saturate(180%); */
```

## 🎓 学到的设计原则

### 1. 色彩选择
- 勃艮第红 = 优雅、成熟、高端
- 黄金色 = 温暖、贵气、强调
- 白色 = 现代、简洁、专业

### 2. 动画设计
- Cubic Bezier 曲线 = 超越终点的弹性
- 0.4秒 = 平衡点（不快不慢）
- 毛玻璃 = 深度、质感、现代感

### 3. 排版
- 系统字体优先 `-apple-system`
- 字距优化 letter-spacing: -0.3px
- 行高平衡 1.6-1.8

### 4. 间距
- 卡片间距：2.5rem
- 元素 padding：2.5rem
- 部分 margin：100px
