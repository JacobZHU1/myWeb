## 🛠️ 基础定制

### 改变主色调

**步骤 1**：打开 `assets/styles.css`

**步骤 2**：找到最开始的颜色定义（第 10-20 行）

```css
:root {
    --primary-color: #6b1d2b;    /* ← 改这个 */
    --primary-light: #8b2c42;
    --primary-dark: #4a1220;
    --accent-color: #d4a574;      /* ← 或改这个 */
}
```

**步骤 3**：改为你喜欢的颜色
```css
/* 例如：深蓝色 */
--primary-color: #1e3a8a;
--primary-light: #3b82f6;
--primary-dark: #1e40af;
```

**步骤 4**：保存，刷新浏览器（Cmd+R）

### 改变动画速度

在 `:root` 中修改过渡时间：

```css
/* 当前：0.4秒 */
--transition: all 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);

/* 改为更快：0.3秒 */
--transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);

/* 改为更慢：0.6秒 */
--transition: all 0.6s cubic-bezier(0.34, 1.56, 0.64, 1);
```

### 调整毛玻璃效果

在 `.glass-effect` 类中：

```css
.glass-effect {
    /* 当前 */
    backdrop-filter: blur(20px) saturate(180%);
    
    /* 更模糊 */
    backdrop-filter: blur(30px) saturate(180%);
    
    /* 更清晰 */
    backdrop-filter: blur(10px) saturate(180%);
}
```

## 🐛 常见问题

### Q: 动画太快/太慢？
**A**: 在 `styles.css` 的 `:root` 中改变 `--transition` 时间值

### Q: 颜色显示不对？
**A**: 1) 清空浏览器缓存（Cmd+Shift+Del）
      2) 硬刷新（Cmd+Shift+R）
      3) 检查 CSS 文件是否保存

### Q: 头像不显示？
**A**: 确保文件路径是 `images/profile.jpg` 且文件存在

### Q: 表单不工作？
**A**: 表单目前是演示模式，需要后端支持（见部署章节）

### Q: 在手机上显示不对？
**A**: 在开发者工具中测试响应式设计，或检查 CSS 媒体查询

## 💡 设计贴士

### 1. 保持一致性
- 所有颜色应来自 CSS 变量
- 所有动画时间统一 (0.4秒)
- 字体大小保持合理比例

### 2. 色彩搭配原则
- 主色 70%（背景、主要元素）
- 强调色 20%（重点、按钮）
- 中性色 10%（文字、分隔线）

### 3. 交互设计
- 所有交互都应有反馈
- 反馈应该立即（< 100ms）
- 使用熟悉的动画模式

### 4. 可读性
- 对比度 > 4.5:1（WCAG AA 标准）
- 字体大小 ≥ 16px（移动设备）
- 行高 ≥ 1.5（文本易读性）
