⏱️ AutoTap Scheduler

A precision web tool I built for split-second actions — set an exact start time like 19:00:00 and it fires a click the instant that moment arrives, useful for grabbing limited tickets or flash-sale items before they sell out.

Scope: a page's JavaScript can never reach into a different tab or site — browsers block that by design (same-origin policy), on iOS, macOS, or anywhere else. So it ships as a bookmarklet that runs directly inside whichever real site you need to act on, since that's the only way to legitimately click something on another site.

Use it on another site

  1. Drag this button to your bookmarks bar: 📌 AutoTap Here
  2. Open the real page with the button you want to click (e.g. a checkout or ticket page).
  3. Click the bookmarklet — a small panel appears on that page.
  4. Click "Pick Target", then click the real button once to mark it (this click won't trigger it).
  5. Set the start time and hit Schedule. Keep that tab open until it fires.
  1. Tap
  2. In Safari, open any page, tap the book icon, then Add Bookmark → Save.
  3. Tap the book icon again → Edit → tap the bookmark you just made.
  4. Clear the address field, tap it, and Paste the copied code, then tap Done.
  5. Open the real ticket/checkout page, then tap that bookmark from your Bookmarks list.
  6. Tap "Pick Target", mark the real button, set the time, and Schedule — keep the tab open until it fires.

This only works while that exact tab stays loaded — closing or refreshing it cancels the schedule, since the timer lives in that page's own JavaScript, nowhere else.
