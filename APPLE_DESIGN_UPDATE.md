## 🔧 技术实现


## 🎯 使用指南

### 1. 本地预览
```bash
# 服务器已启动
http://localhost:9000

# 或手动启动
python3 -m http.server 8000
```

### 2. 修改颜色
编辑 `assets/styles.css` 中的 `:root` 变量：

```css
:root {
    --primary-color: #6b1d2b;      /* 改这里 */
    --accent-color: #d4a574;       /* 改这里 */
}
```

### 3. 调整动画速度
在 `:root` 中修改：

```css
--transition: all 0.3s ...; /* 改为0.3s会更快 */
--transition: all 0.5s ...; /* 改为0.5s会更慢 */
```

### 4. 启用/禁用效果
在 CSS 中注释掉相关的 `backdrop-filter` 行：

```css
backdrop-filter: blur(20px) saturate(180%);
/* -webkit-backdrop-filter: blur(20px) saturate(180%); */
```
